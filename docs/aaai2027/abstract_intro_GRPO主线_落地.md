# 摘要

条件视频世界模型将未来观测编码为离散视觉 token 并自回归生成，可借助全参考评价指标与组相对策略优化（Group Relative Policy Optimization, GRPO）进行后训练。然而，现有基于 GRPO 的世界模型后训练存在两个与其生成结构直接相关的局限。其一，候选未来受冻结 tokenizer 与解码器所定义的可达输出空间约束，而后训练常直接以原始真实帧作为排序参照；真实帧与该可达空间之间的不一致——即目标集合失配（target-set mismatch）——会改变 GRPO 实际消费的组内排序，从而削弱相对学习信号的可靠性。其二，多步生成通常将单个序列级优势均匀广播给全部未来帧的视觉 token，未区分不同时间位置对后续预测质量的差异化贡献。为此，我们提出 RC-GRPO（Reachability-Calibrated Group Relative Policy Optimization），在保持世界模型、视觉 tokenizer 与解码器冻结的前提下，从评价器校准与时间信用分配两个维度重构后训练信号，分别修正 GRPO 的组内相对奖励与时间分辨优势，而不改动组相对归一化形式本身。在评价侧，我们先以可达性审计量化原始目标与可达重建目标造成的候选排序差异，再使用重建校准评价器修正训练期比较参照，并以原始目标约束将校准更新限制在保持保真性优化方向的半空间内；在优化侧，我们以每个未来帧对应的完整视觉 token 块为时间责任单元，为其累计从当前位置到序列末端的逐帧回报并在组内标准化，使信用分配与自回归时间依赖保持一致。在固定 RT-1 协议下，完整 RC-GRPO 相对冻结 base 将 LPIPS 与 MSE 分别降低约 7.6% 与 12.4%；消融表明可达性校准与帧块时间信用分配分别改善 GRPO 的排序输入与时间分辨优势。

关键词：视频世界模型；组相对策略优化；可达性校准；目标集合失配；时间信用分配

# 引言

条件视频世界模型已成为具身人工智能系统的重要基础能力，使智能体能够预测动作执行后的未来视觉状态，从而支撑规划、仿真与决策等核心任务[117, 121, 87, 103, 126, 245, 264]。近年来，大量研究通过离散或连续隐空间学习紧凑的视频世界模型，以兼顾生成质量与计算效率[212, 18, 266, 274]，并进一步利用后训练（post-training）比较多条候选未来轨迹与目标参照，从而优化生成策略。其中，以全参考指标为可验证奖励、以组相对策略优化（Group Relative Policy Optimization, GRPO）更新生成策略的后训练范式尤为关键[302, 17, 18, 303]。然而，这类比较优化的有效性高度依赖评价信号能否准确反映候选未来之间的质量差异——对 GRPO 而言，即组内相对奖励与时间分辨优势是否与模型的生成结构一致。对于采用冻结 tokenizer 和解码器的 tokenized 世界模型而言，真实观测通常无法被完全表示为解码器可生成的输出，因此真实数据分布与模型实际可达输出空间之间存在固有偏差[49, 261]。在长时程视频生成过程中，该偏差可能进一步影响候选未来排序的可靠性，并削弱后训练优化信号的有效传播。因此，条件视频世界模型的 GRPO 后训练设计不仅需要关注优化算法本身，还应重新审视评价参照的构建方式，以及训练信号在时间维度上的分配机制[96, 247]。

针对上述挑战，现有研究大致可以分为三类。第一类方法通过改进 tokenization 或表征学习提高隐空间保真度，以缩小编码目标与解码器输出之间的差距[85, 189, 130, 279]。这些方法能够提升视觉表示能力，但由于冻结解码器所定义的输出空间受限于模型容量与表示方式，真实观测仍可能位于该可达空间之外。第二类方法通过序列级优化策略改善长时程生成中的时间信用分配，包括轨迹重加权、事后重标注以及回报分解等[96, 90, 21]，但部分方法仍依赖粗粒度的序列级归因，难以刻画自回归生成中早期预测对后续视觉结果的因果影响[96, 141, 268]。第三类方法通过奖励塑形或辅助目标增强训练信号[61, 69, 78, 281]，能够缓解稀疏或失真的评价反馈，但通常需要引入任务相关先验，且可能引入额外优化偏差。上述路线分别触及表示、序列归因与奖励设计，却尚未从 GRPO 后训练所依赖的组内排序与优势分配接口给出统一修正。

近期，GRPO 被引入世界模型后训练，通过比较同一候选组内不同未来轨迹的相对质量来优化生成策略[302, 17, 18, 303]。然而，现有基于 GRPO 的世界模型后训练方法仍存在两个关键局限。首先，部分方法直接采用原始真实帧作为排序参照，而生成模型的候选结果通常位于冻结 tokenizer–decoder 定义的可达输出空间内，真实帧与该空间之间的不一致可能导致组内排序偏差，使 GRPO 消费的相对奖励不可靠。其次，现有优势估计通常以序列级信号广播至所有未来视觉 token，而没有考虑自回归生成中不同时间位置对未来预测质量的差异化贡献，使 GRPO 的优势分配与时间因果结构错位。这种由真实目标与可达生成目标之间的不一致所导致的评价偏差，本文称为“目标集合失配”（target-set mismatch）[49, 260]。尽管已有研究分别从表示学习、奖励设计和序列优化等角度进行了探索，但能够同时修正 GRPO 排序输入与时间分辨优势的统一后训练框架仍然缺失。

为解决上述问题，我们提出 RC-GRPO（Reachability-Calibrated Group Relative Policy Optimization）。如图1所示，RC-GRPO 从评价器校准和时间信用分配两个维度重新设计后训练信号，分别对应上述两个关键局限：前者修正 GRPO 的组内相对奖励，后者重构 GRPO 的时间分辨优势。不同于已有方法主要关注提升 tokenizer 表示能力[85, 189, 130, 279]，或依赖额外辅助损失与任务特定奖励塑形[61, 69, 78, 281]，RC-GRPO 在保持冻结 tokenizer 和解码器不变的情况下，通过可达性校准重新构建评价参照，并通过结构化时间信用分配使优化过程与自回归依赖关系保持一致。该方法能够直接适配现有冻结视觉接口的 GRPO 后训练流程，仅利用候选组内部的相对排序信息完成优化，无需额外人工监督信号或改变环境交互协议。

针对评价目标偏差问题，我们首先提出可达性审计（reachability audit），用于量化真实目标与可达重建目标之间的排序差异，从而诊断 GRPO 组内排序是否被错误奖励扭曲。具体而言，我们测量将原始帧替换为其冻结 tokenizer–decoder 重建结果后，候选组排序的一致性变化与候选对翻转比例。在此基础上，构建重建校准评价器，以真实帧经冻结 tokenizer–decoder 映射后的可达重建结果作为排序参照，同时保留 MSE 与 LPIPS 指标，以修正 GRPO 的相对奖励输入。然而，直接替换排序参照可能改变原始真实目标所定义的优化方向。为此，我们进一步引入原始目标约束：分别构造原始目标与校准目标对应的组相对代理目标，并将校准梯度投影至保持原始目标一阶优化进度的约束半空间，使模型在利用可达目标修正排序偏差的同时，保持与真实观测保真性方向的一致性。

针对时间信用分配问题，我们提出视觉 token 块级时间信用分配机制。不同于现有 GRPO 将序列级优势均匀广播至所有未来帧 token 块[302, 17, 18]，我们以每个未来帧对应的完整视觉 token 块为时间责任单元，为第 t 个 token 块累计从第 t 帧至序列末端的逐帧奖励，并在相同上下文和位置条件下进行组内标准化，仅将对应优势作用于第 t 个 token 块。该设计改写的是 GRPO 的优势在时间轴上的作用范围：早期 token 块能够获得来自未来预测的累计反馈，而非假设所有时间位置具有相同贡献，从而使信用分配结构更符合自回归生成过程中的因果依赖关系。

综上，RC-GRPO 形成了一套面向 GRPO 后训练的统一优化框架：在模型可达空间内建立可靠的候选比较机制，通过原始目标约束保持优化方向一致性，并依据自回归时间依赖完成结构化的信用分配。该框架区别于将现成视觉评价指标与通用 GRPO 简单组合的策略[302, 17, 18, 303]，能够同时缓解目标集合失配与长时程信用分配问题，且仍保留 GRPO 的组相对归一化形式。本文的主要贡献如下：（1）针对 GRPO 组内排序依赖评价参照的问题，我们指出 tokenized 视频世界模型后训练中的目标集合失配，并提出可达性审计与重建校准评价器，在可达空间内修正 GRPO 的相对奖励信号。（2）针对序列级优势广播与自回归结构不一致的问题，我们提出视觉 token 块级时间回报，重构 GRPO 的时间信用分配。（3）在固定 RT-1 协议下，完整 RC-GRPO 相对冻结 base 将 LPIPS 与 MSE 分别降低约 7.6% 与 12.4%；跨编解码器（CNN-FSQ、压缩 FSQ 与 NVIDIA Cosmos DV-FSQ[306]）的编解码重建误差与 RT-1/VP2-RoboSuite[110, 111] 的同候选排序审计支持该问题的一般性；消融验证可达性校准与时间信用分配均贡献于 GRPO 后训练增益。
