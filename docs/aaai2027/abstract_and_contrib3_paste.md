## 摘要（整段）

动作条件视频世界模型将未来观测压缩为视觉 token 后进行自回归生成，并可借助全参考评价指标和组相对策略优化进行后训练。然而，可计算的图像误差并不必然构成可靠的相对学习信号：冻结视觉编解码器把候选限制在其可重建输出集合内，而原始真实帧通常不能被该接口精确复现；由此产生的重建残差会与各候选误差发生不同交互，从而改变 GRPO 实际消费的组内排序。多步生成又常把一个序列级优势广播给全部未来帧视觉 token 块，忽略早期预测对后续误差的累积影响。为此，我们提出 RC-GRPO（Reconstruction-Calibrated Group Relative Policy Optimization）。该方法首先在完全相同的候选组上审计原始目标与编解码重建目标造成的秩相关变化和候选对翻转，随后使用重建校准评价器修正训练期比较参照，并以原始评价器约束校准更新的一阶方向；在多步阶段，则以完整帧 token 块为责任单元，将从当前位置到序列末端的逐帧回报分配给能够影响这些结果的前序块。实验上，证据沿机制与预测质量两条线索展开。在 CNN-FSQ、压缩 FSQ 与 NVIDIA Cosmos DV-FSQ 三种独立编解码器上，真实帧经 encode–decode 后的重建残差均不可忽略；在 RT-1 与 VP2-RoboSuite 的同候选审计中，以编解码重建结果为比较目标后，秩相关提高、候选对翻转减少，表明目标侧重建残差会干扰组相对学习信号，且该现象不局限于单一 tokenizer。在固定 RT-1 协议、相对同一冻结 base 时，完整方法将 LPIPS 与 MSE 分别降低约 7.6% 与 12.4%，并优于同协议下的官方 RLVR 读数；长序列生成收益集中在具有预测 headroom 的设置。

## 贡献（1）

（1）我们揭示并操作化了 tokenized 视频世界模型后训练中的目标集合失配：目标侧重建残差并非一定会被组内中心化消除，其候选相关交互会直接改变 GRPO 消费的相对排序。我们据此提出以同候选秩相关、候选对翻转率和表示一致性为核心的可达性审计，把有损编解码带来的重建残差从一般压缩现象转化为可检验的策略优化诊断。

## 贡献（2）（保持方法表述即可）

（2）我们提出 RC-GRPO：在不改动世界模型及冻结视觉接口的条件下，以重建校准评价器修正训练期比较参照，以原始目标约束更新方向，并以未来帧 token 块上的时间回报完成长序列信用分配，从而统一处理候选排序可靠性与自回归时间责任。

## 贡献（3）

（3）我们在跨编解码器和跨世界模型的协议下验证上述机制，并在固定 RT-1 协议下报告相对同一冻结 base 的预测质量。在 CNN-FSQ、压缩 FSQ 与独立设计的 Cosmos DV-FSQ[306] 上，真实帧的 encode–decode LPIPS 重建误差分别约为 0.053、0.077 和 0.190；RT-1[110] 与 VP2-RoboSuite[111, 304] 的同候选审计均显示：以编解码重建结果为训练期比较目标后，秩相关提高、候选对翻转减少。在 episode-disjoint 评测中，完整 RC-GRPO 相对冻结 base 将 LPIPS 与 MSE 分别降低约 7.6% 与 12.4%，并优于同协议下的官方 RLVR 读数；相对序列级重建校准的进一步增益在正文消融中给出。上述结果支持机制诊断与 RT-1 上的多步保真改善，但不构成跨平台原始像素收益的普适主张。
