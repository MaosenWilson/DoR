## 推荐摘要全文（可直接替换）

动作条件视频世界模型将未来观测压缩为视觉 token 后进行自回归生成，并可借助全参考评价指标和组相对策略优化进行后训练。然而，可计算的图像误差并不必然构成可靠的相对学习信号：冻结视觉编解码器把候选限制在其可重建输出集合内，而原始真实帧通常不能被该接口精确复现；目标侧重建残差与候选误差的交互会改变 GRPO 实际消费的组内排序。多步生成又常把一个序列级优势广播给全部未来帧视觉 token 块，忽略早期预测对后续误差的累积影响。为此，我们提出 RC-GRPO（Reconstruction-Calibrated Group Relative Policy Optimization）。该方法首先在完全相同的候选组上审计原始目标与可重建目标造成的秩相关变化和候选对翻转，随后使用重建校准评价器修正训练期比较参照，并以原始评价器约束校准更新的一阶方向；在多步阶段，则以完整帧 token 块为责任单元，将从当前位置到序列末端的逐帧回报分配给能够影响这些结果的前序块。实验上，证据分两层呈现：在 CNN-FSQ、压缩 FSQ 与 Cosmos DV-FSQ 上均观测到不可忽略的编解码重建误差，并在 RT-1 与 VP2-RoboSuite（及跨架构设置）的同候选审计中，一致观察到校准后秩相关提高、成对翻转减少，表明重建残差会腐蚀组相对学习信号且不局限于单一 tokenizer；在固定 RT-1 多步协议下，完整方法相对序列级重建校准进一步降低完整序列 LPIPS、末帧 LPIPS 与 MSE（约 1.4%、1.7% 与 3.2%）。长序列生成收益集中在具有预测 headroom 的设置，近天花板平台用于界定适用范围。

## 仅末段 · 三档观感（数字同源 formal）

### A. 推荐（干净、两层、无 seed）

实验上，我们将证据分为两层。重建侧：CNN-FSQ、压缩 FSQ 与 Cosmos DV-FSQ 均呈现不可忽略的编解码重建误差；在 RT-1 与 VP2-RoboSuite 的同候选审计中，可达目标重评分一致提高秩相关并降低成对翻转，表明目标侧残差会腐蚀 GRPO 排序，且该现象跨 tokenizer 与生成架构复现。多步侧：在固定 RT-1 协议下，帧块时间回报相对序列级重建校准，将完整序列 LPIPS、末帧 LPIPS 与 MSE 分别降低约 1.4%、1.7% 与 3.2%。改进主张限定在具有预测 headroom 的长序列设置。

### B. 更短（字数紧时）

跨三种独立 FSQ 编解码器均存在不可忽略的编解码重建误差；RT-1 与 VP2 的同候选审计一致显示校准提升秩相关、抑制成对翻转。固定 RT-1 多步协议下，完整 RC-GRPO 相对序列级重建校准将 LPIPS、末帧 LPIPS 与 MSE 降低约 1.4%、1.7% 与 3.2%；收益集中在有 headroom 的设置。

### C. 略强 C1（带量级，仍无 seed）

在三种独立编解码器上编解码重建误差均不可忽略；同候选审计中，RT-1 压缩 FSQ 设置下秩相关平均提升约 0.015、成对翻转下降，VP2 上秩相关提升更为显著（约 0.07–0.09）。固定 RT-1 多步协议下，完整方法相对序列级重建校准降低 LPIPS / 末帧 LPIPS / MSE 约 1.4% / 1.7% / 3.2%。
