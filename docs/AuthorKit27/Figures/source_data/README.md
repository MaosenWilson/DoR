# Source Data for AAAI-2027 Figures

These files contain the processed source data used to draw the manuscript figures.

## Files
- `fig_rank_flip_source.csv`: per-seed rank-flip diagnostics for Fig. rank-flip.
- `fig_msp_learning_curves_source.csv`: per-run post-training curves and eval-only baselines for Fig. learning curves.
- `fig_horizon_delta_source.csv`: per-seed, per-horizon values for sequence-level RC and temporal-return RC.

## Provenance
The CSV files were generated by `scripts/figures/plot_aaai_results.py` from audited JSON files under `tmp/aaai_results_raw/`, which mirror server outputs from `/root/autodl-tmp/vote2world/outputs`.

The five raw sequence-GRPO runs and the five calibrated runs are all present locally. The same-candidate RC diagnostic is stored separately as `tmp/aaai_results_raw/analysis/rc_flip_closure.npz`; its 920 rows comprise 184 contexts repeated under five independent candidate-generation seeds and must not be analyzed as 920 independent contexts.

## Variables
- `step`: post-training step.
- `horizon`: rollout horizon used in multi-step evaluation.
- `delta_return_minus_seq`: `return_rc - seq_rc`; negative is better for LPIPS/MSE/MAE, positive is better for PSNR/SSIM.
- `predicted_flip`: theoretical pairwise flip rate `arccos(rho)/pi`.
- `observed_flip`: empirical pairwise rank-flip rate.
- Learning-curve bands and tabulated uncertainty use sample standard deviation (`ddof=1`).

## Access and licence
For submission, these processed figure source data can be included as source data or supplementary files. Raw model outputs remain in the project `outputs/` directory.
